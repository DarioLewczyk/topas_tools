# topas_tools
topas_tools is a complete tool that can fully automate Rietveld refinements for in situ experiments performed at the NSLS-II and provide you with high quality graphics based on the output of TOPAS's refinements. 

## Basic usage:
* To refine patterns using topas_tools, import "TOPAS_Refinements" from "refine.py".
* To analyze your data after you complete refinement, import "RefinementAnalyzer" from "refinement_analyzer.py"

All of the classes have extensive commentation and docstrings which explain their functionality.
